导入说明

1. 升级 hydrooj 至最新版本（至少 4.10.2，更新说明 https://docs.hydro.ac/FAQ/#%E6%9B%B4%E6%96%B0%E5%8D%87%E7%BA%A7 ）。
2. 使用 hydrooj install https://hydro.ac/hydroac-client.zip 安装相关模块。
3. 重启 hydrooj 。
4. 使用 【从 Hydro 导入】功能导入该题库。
5. 重启 hydrooj 。

至此，题目已经可以正常提交了。


注：此功能不使用 ssoier 提供的评测接口，无需申请教师账号，可直接使用。
